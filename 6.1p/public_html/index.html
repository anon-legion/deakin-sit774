Creating a list of Butterflies using JavaScript

We all love Butterflies and we also know how important JavaScript is to supporting modern web development. On this page we combine both of these to demonstrate how to dynamically create content and insert it into the web pages' DOM (Document Object Model). In this case, the visitor's name is requested, and an ordered list is dynamically created; showing the names of 20 Butterflies from the United Kingdom [Source: https://butterfly-conservation.org/red-list-of-butterflies-in-great-britain].
